# 🍿 Crunchy Bites - Premium Namkeen Website

A modern, responsive and visually appealing **Namkeen (Snacks) Website** built using **HTML, CSS, and JavaScript**.
This project showcases a premium Indian snacks brand with smooth UI, animations, and interactive features.

## ✨ Features

* 🏠 Beautiful Hero Section with Auto Slider
* 🧭 Sticky Navigation Bar
* 🛍️ Product Showcase Grid
* 🛒 Add to Cart Interaction (UI Based)
* ⭐ Customer Testimonials Section
* 📩 Contact Form with Validation
* 📱 Fully Responsive Design (Mobile Friendly)
* 🎨 Smooth Animations & Hover Effects
